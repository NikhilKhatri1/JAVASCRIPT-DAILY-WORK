1. Date Object and Constructors

            1. To create a Date object with the current date and time:
                Syntax:-
                    var now = new Date();

            2. You can also create a Date object by specifying a date and time string:
                Syntax:-
                    var customDate = new Date("2024-06-28 12:30:00");

            2. Methods to Get Date and Time Components
            
            The Date object provides various methods to Retrieve different components of the date and time:
            
            getHours(), getMinutes(), getSeconds(), getMilliseconds(): Retrieve hour, minute, second,
            and millisecond components respectively.
            getDay(): Get the day of the week (0 = Sunday, 1 = Monday, ..., 6 = Saturday).
            getMonth(): Get the month (0 = January, 1 = February, ..., 11 = December).
            getDate(): Get the day of the month (1 to 31).
            getFullYear(): Get the full year (e.g., 2024).
            toDateString(): Returns "Wed Jun 28 2024" without Time
            toLoacleDateString(): Returns localized date string (e.g., "6/28/2024" in US locale).
            toTimeString(): Returns time string (e.g., "14:30:00 GMT+0300 (Eastern European Summer Time)").
            tolocaleTimeString(): Returns localized time string (e.g., "2:30:00 PM" in US locale).

            3. Methods to Set Date and Time Components
            
            JavaScript also provides setter methods to modify these components dynamically:
            
            setHours(), setMinutes(), setSeconds(), setMilliseconds(): Set the hour, minute,
            second, and millisecond components respectively.
            setDate(), setMonth(), setFullYear(): Set the day of the month, month, and full year respectively.
