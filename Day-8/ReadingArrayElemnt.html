1. Reading Array Elements
            
                    toString() : It returns all elements separated with "," delimiter.

                    join() : It returns all elements with custom delimiter.

                    map() : It is an iterator to read all elements with custom presentation.

                    forEach() : It is an iterator to read all elements with index number.

                    for..in : It is an iterator to read all properties

                    for..of : It is an iterator to read all values

                    find() : It returns the first occurrence value that matches give
                    condition.

                    filter() : It returns all values that match given condition.
