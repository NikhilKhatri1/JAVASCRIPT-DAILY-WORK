Number Types

  1. Signed Integer=(-34)
  2. UN-Signed Integer=(34)
  3. Floating point = (34.45)
  4. Double = 450.453
  5. decimal = 3400.44
  6. Exponent = 2e3 = 2*103 = 2000
  7. Binary = 0b1010
  8. Hexadecimal = 0x4499
  9. Octa = 0o576
  10. Bigint = 9849582398n (to tell the compiler that this number is safe)

To convert the String into number

  1. parseInt()
    2. Syntax
    parseInt("24A") // 24
    it will convert 24 into number and any number afer that will be removed
  2. parseFloat()
    3. Syntax
    parseFloat("24.45A") // 24.45
    It will convert into float value
