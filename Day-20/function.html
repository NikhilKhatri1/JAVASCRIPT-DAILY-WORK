        Function Declaration
        
        Function declarations are defined using the function keyword. They are hoisted to the top of their scope, meaning you
        can call the function before it's declared in your code.
        
        Example:
        
        javascript
        
        PrintMessage(); // You can call it before declaration
        
        function PrintMessage() {
        console.log("Hello, this is a function declaration.");
        }
        
        In this example, PrintMessage is declared using the function declaration syntax. It can be called anywhere within the
        same scope, even before its actual declaration due to hoisting.
        Function Expression
        
        Function expressions involve assigning a function to a variable. They are not hoisted like function declarations, so
        they cannot be called before the variable assignment.
        
        Example:
        
        javascript
        
        Print(); // This would throw an error: Uncaught TypeError: Print is not a function
        
        var Print = function() {
        console.log("Hello, this is a function expression.");
        };
        
        In this example, Print is assigned an anonymous function using a function expression. If you try to call Print() before
        the assignment (var Print = ...), it will result in an error because Print is not yet defined as a function.
