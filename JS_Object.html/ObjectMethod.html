Object Methods

A method is a function definition stored as a property value.

Accessing Object Methods

            You access an object method with the following syntax:
            
                    objectName.methodName()

Note:- If you call the function with () it will execute as a function...
document.getElementById("demo").innerHTML = person.fullName();
O/p:- John doe
if you call the function without () it will return function defination..
document.getElementById("demo").innerHTML = person.fullName;
O/p:- fullName: function () { return this.firstName + " " + this.lastName; // "this" refers to the person object: }
